# LiberWiki – The Freedom Encyclopedia

**LiberWiki – The Freedom Encyclopedia** es un proyecto personal experimental cuyo objetivo principal es servir como **prueba técnica y conceptual para la creación de una enciclopedia personal tipo Wikipedia utilizando MkDocs y GitHub Pages**, sin costes y sin infraestructura de servidor.

El contenido del sitio está centrado en el pensamiento libertario, pero el propósito del repositorio es tanto **técnico** como **editorial**: explorar hasta qué punto MkDocs puede emplearse como motor para una enciclopedia estática, estructurada y académica.

El sitio es accesible públicamente en:  
https://dan1max.github.io/liberwiki/

---

## Objetivo del Proyecto

Este repositorio existe como:

- Una **prueba funcional** de MkDocs como alternativa ligera a MediaWiki (el motor de Wikipedia).
- Un experimento de **“Wikipedia personal” estática**, sin bases de datos ni backend.
- Un banco de pruebas para:
  - organización enciclopédica del conocimiento,
  - navegación temática,
  - estilos académicos clásicos,
  - control de versiones mediante GitHub.

No se trata de un producto comercial ni de una plataforma colaborativa masiva, sino de un **experimento controlado y reproducible**.

---

## Tecnologías Utilizadas

- **MkDocs** como generador de sitio estático
- **Markdown** como formato de contenido
- **GitHub Pages** como sistema de alojamiento
- **CSS personalizado** para tipografía clásica (Times New Roman) y esquema de colores

Todo el proyecto se gestiona directamente desde la interfaz web de GitHub.
